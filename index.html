STEP3

코드 생성 요청

구글 앱스크립트 형식에 맞게 완성된 코드를 받아요. 이 프롬프트가 핵심이에요!

약 2분
AI에게 보낼 프롬프트 (④조건 + ⑤형식 — 가장 중요!)
💬 반드시 아래 조건을 포함해서 요청하세요
앞에서 정리한 구성 그대로 구글 앱스크립트 코드를 완성해줘.

반드시 다음 조건을 지켜줘 (④조건):
1. doGet() 함수 하나 안에 HTML, CSS, JavaScript를 모두 넣기
2. HtmlService.createHtmlOutput(html) 으로 반환하기
3. 화면 텍스트는 한국어로 만들기
4. 버튼 클릭·저장 처리 중에는 "처리 중..." 로딩 메시지 표시
5. 스마트폰에서도 잘 보이는 반응형 디자인 (max-width, flexbox 활용)
📎 구글 시트 연동이 필요하다면:
6. SpreadsheetApp.openById(SHEET_ID) 방식, SHEET_ID는 코드 맨 위에 선언
⚡ 데이터를 실시간으로 자동 새로고침하고 싶다면:
7. setInterval + google.script.run 방식으로 주기적으로 서버에서 데이터 가져오기

⑤형식: 코드 블록으로 전체 코드만 먼저 주고, 설명은 코드 아래 3줄 이내로.
AI가 코드를 주면 전체 코드를 복사해두세요. 다음 단계에서 바로 붙여넣을 거예요!
STEP4

코드 붙여넣기 & 배포

받은 코드를 구글 앱스크립트에 붙여넣고, 웹앱으로 배포해서 링크를 받아요!

약 5분
✅
개인 gmail 계정도 OK! — 교육청 전용 계정이 없어도 됩니다.
배포 시 액세스 권한을 "모든 사용자"로 선택하면 누구나 접속 가능한 링크가 생성됩니다.
교육청 계정이 있다면 "도메인 내 사용자"로 제한할 수도 있어요.
앱스크립트 접속 → 배포 (6단계)
1
크롬 주소창에 script.google.com 입력 → 구글 계정 로그인
2
"새 프로젝트" 클릭 → 기존 코드 전체 선택 Ctrl+A → 삭제
3
AI가 준 코드 전체 붙여넣기 Ctrl+V → 저장 Ctrl+S
4
오른쪽 위 "배포" → "새 배포" → 유형: 웹 앱 선택
5
액세스 권한: 모든 사용자 선택 → "배포" 클릭
6
웹 앱 URL 복사! 🎉 — 스마트폰으로 접속해서 확인해보세요
처음 배포 시 구글 권한 허용 팝업이 뜰 수 있어요. "계속" → "허용" 클릭하면 됩니다. 코드 수정 후 재배포할 때는 "새 버전"으로 다시 배포하세요.
STEP5

오류 수정

오류가 생겼나요? 걱정 말고 오류 메시지를 AI에게 그대로 보내면 돼요.

필요할 때만 진행
오류 발생 시 — AI에게 보낼 프롬프트
💬 오류 메시지를 그대로 붙여넣어요
다음 오류가 발생했어요:

[오류 메시지를 그대로 붙여넣기]

어떻게 수정하면 될까요? 수정된 전체 코드를 다시 주세요.
자주 발생하는 오류 유형
오류 1

배포 후 빈 화면
→ doGet() 함수 이름 확인. 철자 오류 주의.

오류 2

권한 오류(Permission)
→ 배포 시 액세스 권한 "모든 사람" 재확인.

오류 3

시트 저장 오류
→ SHEET_ID가 비어있거나 잘못됨. URL에서 ID 재확인.

오류 4

화면이 깨져 보임
→ CSS가 빠졌을 수 있음. AI에게 전체 코드 다시 요청.

기존 기능은 그대로 유지하고, 수정된 전체 코드를 다시 주세요.
이런 기능을 추가해볼 수 있어요 (설문 응답 기반)
"학생이 이름을 입력하면 구글 시트에 자동으로 저장되게 해줘"
"모바일에서도 잘 보이게 디자인을 개선해줘"
"입력 후 확인 알림(토스트 메시지)을 추가해줘"

바이브코딩 핵심 원칙: 코드를 이해하지 않아도 돼요. 원하는 기능을 말로 설명하면 AI가 코드로 바꿔줍니다. Claude, ChatGPT, Gemini 모두 가능!
